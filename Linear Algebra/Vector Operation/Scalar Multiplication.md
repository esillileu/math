# Scalar Multiplication
> [[Vector|벡터]]의 각 [[Vector Component|성분]]에 대한 스칼라의 곱
## 정의 
+ ***스칼라 곱***은 스칼라 $c$와 벡터 $\mathbf v$의 각 [[Vector Component|성분]]을 곱하는 [[Binary Operation|이항연산]]
	+ $$\begin{matrix}
		c \in \mathbb R \\
		\mathbf v = (v_1, v_2, v_3, \cdots) \\ 
		c \times \mathbf v = (c\times v_1,\; c\times v_2,\; c\times v_3, \cdots)
		\end{matrix}$$
