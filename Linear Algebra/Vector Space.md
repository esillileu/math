# Vector Space
> [[Field|체]]위의 [[Module|가군]]
## 정의
+ ***[[Field|체]] $(F, +, \cdot)$ 위의 벡터공간 $V$*** 는 [[Field|체]] $(F, +, \cdot)$위의 [[Module|가군]]
	+ ***스칼라***는 벡터공간 $V$의 체 $(F, +, \cdot)$  
	+ ***영벡터***는 [[Abelian Group|가환군]] $(V, +)$의 [[Binary Operation#항등원|항등원]]
	+ [[Vector|벡터]]는 벡터공간 $V$의 원소
